1.根据地址修改,index.php 和server.php文件  ,其中端口你可以指定(不要与服务器http相同)
2.通过命令行执行php文件  如 php -q c:\path\server.php  
3.浏览器(支持html5)访问index.php 页面,   此连接可测试浏览器是否支持Websocket http://www.websocket.org/echo.html

注意:
php 已经加载socket模块


Running Server :

1. Change host address in index.php and server.php

2. Go to your shell command-line interface

3. type: 
	php -q c:\path\server.php

4. Using browser, navigate to index.php location to open chat page, have fun!
